# TurtleScript
TurtleScript is a scripting language I made to control Python's turtle module


Here's an example
```
# simple script that draws a square

# create a new turtle named 't'
new turtle t


# move the turtle

t forward 100

t left 90

t forward 100

t left 90

t forward 100

t left 90

t forward 100

t left 90
```
